# create-go-app

Create Go apps with with zero configuration by using a single command.

#### Table of Contents

- [Installation]()
- [Creating an App]()

## Installation

    go get -u github.com/andygeiss/create-go-app

## Creating Go Apps

Create a Go binary:

    create-go-app -type bin -name <name>

Create a Go library:

    create-go-app -type lib -name <name>
