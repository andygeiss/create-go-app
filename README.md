# create-go-app

Create a microservice in Go by using a single command.

#### Table of Contents

- [Installation]()
- [Creating an App]()

## Installation

    git clone https://github.com/andygeiss/create-go-app
    cd create-go-app
    make

## Creating Go Apps

Create a Go binary:

    create-go-app -type bin -name <name>

Create a Go library:

    create-go-app -type lib -name <name>
